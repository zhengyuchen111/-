学生信息管理系统项目介绍
本系统是面向学校教务场景的轻量化信息管理工具，核心解决传统学生信息管理效率低、权限混乱等问题，分为用户（学生） 和管理员（教务） 两大登录体系：
学生端：支持用户名密码登录，可注册账号、忘记密码、查询账号操作记录；
管理员端：多权限账号登录，可完成学生信息的录入 / 查询 / 修改 / 删除、账号管控、数据统计导出。
系统采用密码加密、操作日志等安全机制，适配 PC 端使用，能显著提升教务管理效率，保障数据安全，实现学生信息规范化、数字化管理。
使用说明
学生信息管理系统分为用户登录和管理员登录
用户登录输入自己的用户名或者邮箱然后还有密码进行登录，用户登录进去可以查看个人信息，我的成绩，个人信息，我的课程，退出登录还有成绩排名登，课程主要有高等数学，大学物理，程序设计基础，思想政治等。个人信息可以进行个人信息绑定还可以解除绑定等。
管理员登录（账号：admin 密码：admin123）
管理员登录进去学生管理，成绩管理和课程管理，学生管理可以添加和删除学生，课程管理可以对课程进行添加和删除功能，成绩管理可以修改和删除学生成绩，管理员总体来说是进行增删改查
